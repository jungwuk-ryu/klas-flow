# AGENTS.md

## 작업 규칙

- 본 저장소는 고수준 SDK 구현에 집중한다.
- 로우레벨 엔드포인트 세부사항은 외부 문서/코드에 노출하지 않는다.
- 비공개 명세 문서는 절대 저장소에 커밋하지 않는다.

## 보안/테스트 규칙

- 실계정으로 테스트할 때 상태 변경 API(게시글 등록/수정/삭제 등)를 호출하지 않는다.
- 기본 테스트는 Mock HTTP 기반 단위 테스트로 구성한다.
- 통합 테스트를 추가할 경우 읽기 전용 API만 사용한다.

## 코드 규칙

- 주석/문서는 자연스러운 문장으로 작성한다. 용어가 어색하면 영어를 사용한다.
- `lib/src`는 내부 구현 영역으로 유지하고 공개 API는 `lib/klasflow.dart`, `lib/klas_client.dart`로 제한한다.
