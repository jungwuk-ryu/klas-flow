# Roadmap

## Near-Term (v1.x)

- 도메인 API endpoint별 강타입 모델 확대
- `runHealthCheck()` 실패 패턴별 대응 가이드 자동화
- Flutter 샘플 앱(로그인/목록/상세) 추가
- API 응답 스냅샷 기반 회귀 테스트 강화

## Mid-Term (v2)

- API drift 감지 리포트 자동 생성(`tool/`)
- observability hook(요청/응답 이벤트, 민감정보 마스킹 내장)

## Adoption Goals

- 신규 개발자 30분 온보딩
- KLAS API 변경 탐지 평균 시간 1일 이내
- 릴리스 전 품질 게이트 100% 자동화
