# klasflow 위키

`klasflow`를 처음 쓰는 개발자를 위한 한국어 튜토리얼 위키입니다.  
이 위키는 "저수준 엔드포인트 호출"이 아니라 "고수준 SDK 객체를 실제 앱에서 어떻게 쓰는지"에 집중합니다.

## 먼저 읽기

- 이 프로젝트는 **개발 단계(Work in Progress)** 입니다.
- 아직 **pub.dev 미배포** 상태입니다.
- 실계정 테스트 시에는 반드시 **읽기 전용 시나리오**만 사용하세요.

- 다룸: `KlasClient`, `KlasUser`, `KlasCourse`, `feature` 객체 기반 고수준 API
